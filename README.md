# 🌐 AI Health Companion
AI Health Companion is a Streamlit-based web application that uses Google Gemini AI to generate personalized health, nutrition, and workout guidance.
The application provides structured meal plans, food image analysis, health insights, workout routines, and progress tracking in one integrated platform.

## 🚀 Features
**1️⃣ Personalized Meal Planner**
1. Generates structured 7-day meal plans

2. Includes calories and protein estimates

3. Considers:

   - Health goals

   - Medical conditions

   - Fitness routine

   - Food preferences

   - Dietary restrictions

4. Provides shopping list and meal prep tips

5. PDF download support

**2️⃣ Food Analyzer (Image-Based)**

1. Upload food image (JPG/PNG)

2. AI identifies food items

3. Provides:

   - Estimated calories

   - Macronutrient breakdown

   - Portion estimation

   - Health benefits & dietary concerns

**3️⃣ Health Insights**

1. Ask nutrition-related questions

2. AI provides evidence-based explanations

3. Personalized recommendations based on user profile

**4️⃣ AI Workout Planner**

1. Generates structured 7-day workout plan

2. Based on:

   - Fitness level

   - Location (Home/Gym)

   - Equipment

   - Goal (Fat Loss/Muscle Gain/etc.)

3. Includes sets, reps, rest intervals

4. PDF download support

**5️⃣ Progress Tracker**

1. Track:

   - Daily weight

   - Calories consumed

   - Workout completion

2. Stores data in CSV file

3. Visualizes trends using charts
